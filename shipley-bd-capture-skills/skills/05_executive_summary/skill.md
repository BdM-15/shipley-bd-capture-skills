# Executive Summary

## Purpose

Convert raw findings into a concise executive summary that is useful for capture decisions.

## Default workflow

1. Gather validated findings from upstream skills.
2. Organize them as signal, inference, evidence, and action.
3. Add confidence and unresolved gaps.
4. Format for fast executive review.

## Output

Return a concise summary with key findings, implications, actions, confidence, and gaps.
