# Shipley Milestone 2 Pursuit

## Purpose

Support a pursuit decision with clear evidence, rationale, and unresolved issues.

## Default workflow

1. Review the current opportunity position.
2. Assess strength, timing, and customer alignment.
3. Highlight evidence for and against pursuit.
4. State the pursuit recommendation and gaps.

## Output

Return the pursuit view with recommendation, evidence, confidence, and next steps.
