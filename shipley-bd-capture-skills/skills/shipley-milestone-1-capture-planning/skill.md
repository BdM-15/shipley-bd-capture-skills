# Shipley Milestone 1 Capture Planning

## Purpose

Turn early opportunity understanding into a practical capture planning view.

## Default workflow

1. Gather the current baseline and research outputs.
2. Identify planning priorities and missing intelligence.
3. Propose near-term actions and owners if available.
4. State the planning view and risks.

## Output

Return the milestone planning summary, evidence, confidence, and next actions.
