# Shipley Milestone 0 Qualification

## Purpose

Support an early qualification decision with concise evidence, rationale, and gaps.

## Default workflow

1. Review the current opportunity facts.
2. Assess basic fit, timing, and customer relevance.
3. Separate facts from assumptions.
4. State the qualification recommendation and key gaps.

## Output

Return a qualification view with recommendation, supporting evidence, confidence, and follow-up actions.
