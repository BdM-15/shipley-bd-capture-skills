# Shipley Milestone 3 Bid No-Bid

## Purpose

Frame the bid decision in a way that is evidence-based, explicit, and actionable.

## Default workflow

1. Review the latest opportunity intelligence.
2. Separate must-have evidence from assumptions.
3. Weigh strengths, risks, and blockers.
4. State the bid or no-bid recommendation and rationale.

## Output

Return the recommendation, supporting evidence, confidence, blockers, and follow-up actions.
